Guidelines for the Download data page

General map tools

Whole Finland tool - reset the map window to show all Finland
Pan tool - move the map with dragging
Zoom in and out tool - change map scale. Alternatively use mouse wheel OR double-click and Shift+double-click OR Shift+drag rectangle for zooming.
Layer control - choose which layers are displayed on map, by default are displayed background map, map sheets and if available the selected dataset. Optionally add Finnish municipality borders and drainage areas.

  • Index map in the bottom left corner of the map window shows the area in which you have zoomed in
  • Place name search above the map enables finding globally place names and addresses available in OpenStreetMap.

Download data from Paituli

  1. Select the dataset from the drop-down menus in the upper left corner of Download data page or from Metadata page. Select data producer, dataset, scale, year, format and coordinate system.
    • Note that if a drop-down menu is white, it has several options, if it is gray, it has only one option available for the selected dataset.
    • In the Files for download tab are displayed:
      • Link to license terms
      • Selected files (map sheets) for download
      • If the dataset is divided into map sheets, then first a tip for selecting the map sheets.
    • On the map is displayed the area of selected dataset and its map sheets.
  2. If the dataset is divided to map sheets, select map sheets from the map or use the map sheet search:
    • Choose the map sheets From the map with:
      • Select the download area by point or rectangle tool - choose map sheets with clicking or dragging a rectangle
      • Select the download area by polygon tool - choose map sheets with drawing a polygon
      • If new map sheets are selected from the map they are added to the existing selection.
    • Map sheets search uses the file names of selected dataset and removes earlier selected files from selection.
    • Unselect map sheets:
      • Unselect all map sheets
      • If selecting again an already selected map sheet from the map or the Files for download tab it will be removed from the selection.
  3. Start the data download by clicking the Download dataset as zip file button. Send data download link to my e-mail window is displayed.
  4. Read the terms of data use and accept them
  5. Write your e-mail address
  6. Click Send data download link to my e-mail-button.
    • The service zips the selected files to one file and sends the download link to the given e-mail. The generation time of the download link depends on the size of the selected data and the load of the service, but should normally be a couple of minutes.
    • The most common reasons for not recieving the e-mail are mistakes in e-mail address or sometimes the e-mail might be directed to Spam.

Preview data

Several of the datasets can also be previewed on the Download data page. The data preview is available by default, after the dataset has been selected.

  • Some of the datasets are displayed only in bigger scales, then there is a notice about zooming in in the upper left corner of the map.
  • Change the visibility of the data preview layer with the map layer control tool.
  • Info tool - check the attribute values of vector features or values of raster pixel. The results are displayed on the left in Feature info tab after clicking on the map.
