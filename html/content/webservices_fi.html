Paitulin rajapintapalvelut

Monet Paitulin aineistot ovat saatavilla OGC:n mukaisista rajapintapalveluista: WMS, WCS, WFS, WMTS. Myös uudet OCS API rajapinnat ovat käytettävissä. Rajapinnoista saatavilla olevat aineistot ovat listattuna alla. OGC rajapintapalvelut mahdollistavat datan latauksen suoraan netti-, työpöytäsovellukseen tai aineiston analyysiskriptiin, näin että dataa ei ladata ensin paikalliselle koneelle tiedostona. Tämä mahdollistaa aina ajan tasalla olevan datan käytön. Rajapintapalvelun käyttö on työpöytäsovelluksissa samanlainen kuin paikallisella levyllä olevan datan käyttö, rajapintapalveluun otetaan yhteyttä vastaavasta valikosta. Yhteyden muodostamiseksi tarvitaan palvelun URL osoite. Yhteyden muodostamisen jälkeen käyttäjälle listataan kaikki palvelusta saatavilla olevat karttatasot.

Rajapinta-aineistojen lisenssiehdot ovat vastaavat kuin latauspalvelun kautta ladatuissa aineistoissa. Lisenssiehdot löytyvät Metadata sivulta. Jos Paitulin rajapintoja käytetään nettisovelluksessa myös Paituli on mainittava.

Uudet OGC API rajapinnat

OGC API rajapinta-standardit ovat vielä kehittymässä. Paitulissa OGC API rajapintojen saatavuus riippuu GeoServerin tarjottavista mahdollisuuksista. Myös asiakassovellukset ovat vielä kehittymässä, esimerkiksi QGIS ja ArcGIS Pro tukevat täällä hetkellä vain OGC API Features rajapintaa.

Tyyppi Pyynnön palaute Rajapinnan osoite
Maps, samankaltainen kuin WMS Karttakuva katsomiseksi https://paituli.csc.fi/geoserver/ogc/maps
Tiles, samankaltainen kuin WMTS Karttakuvan tiili katsomiseksi https://paituli.csc.fi/geoserver/ogc/tiles
Features, samankaltainen kuin WFS Vektoriaineisto https://paituli.csc.fi/geoserver/ogc/features

WMTS-palvelussa karttatiilit ovat saatavilla vain etukäteen määritetyissä mittakaavoissa ja laajuuksissa. WMS-palvelussa kartan koko ja mittakaava voidaan valita vapaasti. WMTS-palvelu on yleensä nopeampi, sillä karttatiilet ovat usein jo palvelun välimuistissa valmiina. Karttatasosta voidaan pyytä myös vain osa käyttämällä suodattimia, esim. BBOX määrittele pyydettävän maantietteellisen alueen.

WMTS-palveluiden ruudukot: ERTS-TM35-FIN (EPSG:3067, JHS180) ja Spherical Mercator (EPSG:3857).
