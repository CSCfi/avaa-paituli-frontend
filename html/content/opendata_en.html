Share your spatial data with Paituli

Students and personnel of Finnish universities have possibility to share their own spatial data with Paituli. It could be data that has been gathered for a scientific study or a course project. This data could be about Finland or any other region in the world.

After publishing anybody can download your data. Datasets suitable for GeoServer can also be previewed on download page and they will be available also as standard OGC web services (WMS, WCS, WMTS or WFS).

Annually CSC generates a report about Paituli usage, including number of downloads for each dataset. This report will be sent to all data providers. Each dataset in Paituli has a permanent identifier in URN format.

CSC publishes data free of charge and will not get any rights for the data.

Steps for sharing your data

  1. Make sure the data can be published. You should either own the data or be granted the right to publish the data by the data owner. No contractual or intellectual property rights should prohibit opening the data. The data should be licensed with a license allowing reuse. Paituli recommends Creative Commons BY 4.0 license.
  2. Describe the data. The data description should include:
    • Heading in Finnish and English
    • Short description in Finnish and English (ca 5-10 rows)
    • Longer description can be added as .pdf-file. (Optional)
    • Dataset producer: organization, optinally researcher by name and ORCID.
    • Date, when data was gathered
    • Scale / pixel size
    • Coordinate system
    • License, for example CC-BY-4.0
    • Project, where the dataset was produced: name and web-page. (Optional)
    • DOI, URN or other PID if the dataset has it already (Optional)
  3. Package your data, technical recommendations
    • Format: Compressed Geotiff or JPG2000 for raster, GeoPackage or Shape for vector data, LAZ for laser scanning data.
    • Coordinate system: ETRS-TM35FIN (EPSG:3067) for data about Finland, WGS-84 (EPGS:4326) for global data.
    • Maximum file size 2 Gb.
    • Divide data to map sheets if needed, if you have map sheet index as a separate file, include that to data package
    • Make a .zip file of all the files.
    • These are only recommendations, if you feel some other option would suit your data better, please consult with CSC.
  4. If you want to use a specific map style for visualizing your data in WMS and WMTS services, make a style description file in SLD format and add it to the data package. If you are not familiar with SLD, also styles suitable for ArcGIS or QGIS may be submitted. (Optional)
  5. Send the data and its description to giscoord@csc.fi. Smaller datasets may be sent as attachments, for bigger files FUNET FileSender is better. FileSender can accept files up to 200 Gb. If your data is bigger or you have data in Allas or Puhti services or if you have problems with uploading, please contact CSC.
  6. CSC will publish your data to Paituli in a week or two.

If you have any questions, please contact CSC, giscoord@csc.fi
